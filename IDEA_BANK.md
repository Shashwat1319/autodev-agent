# AutoDev Idea Bank — Saved for Future

> Generated: 20 June 2026
> These ideas are researched and confirmed as **not built yet** in the market.

## Idea 1: DepRisk
**"Credit Score for Your Dependencies"**
- Scans dependencies → assigns ML-based Risk Score
- Predicts abandonment, license changes, security incidents BEFORE they happen
- Auto-generates PRs to swap high-risk deps
- Monetization: $19/mo individual, $99/mo team

---

## Idea 2: DocDebt
**"SonarQube for Documentation"**
- Measures Documentation Debt Score (0–100)
- Tracks: undocumented functions, stale docs, missing examples
- Auto-creates GitHub Issues/PRs to fix doc debt
- "Doc Gates" to block deploys below threshold
- Monetization: Free (public), $29/mo (private), $199/mo (enterprise)

---

## Idea 3: GitGraveyard
**"Where Deleted Code Goes to Rest"**
- Indexes every deleted function/file across repos
- Searchable web UI for ghost code
- One-click restore via PR
- Near-death tracking (branches that almost deleted code)
- Monetization: $12/mo individual, $49/mo team

---

## Idea 4: PR Pulse
**"Team Health from Pull Requests"**
- Analyzes: review time, sentiment, load balance, response time
- Team Health Score with trends
- Slack digest: bottlenecks, friction, bus factor alerts
- Monetization: $99/mo per 10 team members

---

## Idea 5: Context Keeper
**"Never Lose Your Dev Flow"**
- Full workspace state snapshots (files, terminal, breakpoints, git state)
- Save/load contexts with one command
- Auto-save on computer lock (context switch detection)
- Team context sharing
- Monetization: Free (basic), $8/mo Pro, $49/mo Team
